# Simple-Contact-Book-Manager
Data structures and algorithms (DSA) in Python

Fundamental Structures & Searching/Sorting
1. Simple Contact Book Manager
DSA Focus: Lists, Dictionaries, Basic Searching (linear search), Sorting (by name or phone number).

Description: Create a console application that allows users to add, delete, view, and search for contacts. Store each contact as a dictionary within a main list. Implement a function to search contacts by name and another to display all contacts sorted alphabetically.
